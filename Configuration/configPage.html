Language-Aware Images

Replaces the built-in TMDB image provider. By default the plugin uses each library's configured metadata language, then sorts images by vote_count within that language bucket — matching TMDB's own UI sort order.

Leave empty (recommended) to use each library's metadata language. Set a 2-letter code like de or fr to force that language for every library, ignoring per-library settings.
Used when the preferred language has no images. Usually en.
