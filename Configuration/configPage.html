Language-Aware Images

Replaces the built-in TMDB image provider. Uses each library's metadata language by default and sorts images by vote_count within each language bucket — the same way TMDB's own UI ranks them.

Languages

Leave empty (recommended) to use each library's metadata language. Set a 2-letter code like de or fr to force that language for every library, ignoring per-library settings.

Adds a third bucket between preferred and fallback. Useful for anime / foreign-cinema fans: a German user can see the Japanese poster for Princess Mononoke instead of the US export. Costs one extra API call per item.

Allow textless (no language tag) images

Per image type, since logos and backdrops usually work fine textless but posters generally don't.

Quality filters

Drops images with fewer than this many TMDB votes. Default 0 keeps everything — a vote count of 0 usually means "not voted on yet" rather than "junk", and you don't want to fall back to English just because the German poster hasn't been rated. Bump to 1+ to aggressively trim unvalidated uploads.

TMDB API

Leave empty to use Jellyfin's bundled key. Bring your own to avoid sharing rate limits.
